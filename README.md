# Exposition virtuelle – Merveilles de l'Égypte

## Thème
Le site présente une exposition virtuelle sur les merveilles de l'Égypte ancienne : pyramides, temples et éléments de la civilisation égyptienne.

## Structure du site
Le site contient 4 pages :

- **Accueil** : présentation générale de l'exposition
- **Œuvres / Collection** : description de lieux importants comme les pyramides de Gizeh et le temple de Karnak
- **Galerie** : grille d'images liées à l'Égypte
- **Visite & Contact** : informations pratiques, contact et ressources externes

## Technologies utilisées
- HTML
- CSS
- Flexbox
- SVG

## Sources des images
Images utilisées provenant de sources libres :

- Unsplash
- Wikimedia Commons
- National Geographic (références éducatives)

## Compatibilité
Le site est compatible avec **Google Chrome** et ne contient **aucune librairie externe**.
